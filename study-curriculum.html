운영 구조
  • 호스트 (3명) — 커리큘럼 설계, 과제 플래닝, 문의 답변 담당
  • 게스트 (최대 6명) — 호스트 인원의 2배수로 모집, 과제 수행
  • 1:1 코칭 전담 (개발팀) — 호스트(파마브로스 AX 개발팀)에서 게스트별 전담 코치를 배정하여 개인 채널(DM)로 일대일 소통. 막힌 지점·개인 도전 과제 설계·코드 리뷰를 별도 채널에서 빠르게 해결합니다.
  • 오프라인 모임 (1시간) — 지난 과제 리뷰 → Q&A → 다음 과제 플래닝 → 문의 답변
  • 과제 수행 — 모임 후 다음 수업 전까지 개별 과제 진행
  • 개인채널 연계 — 과제 결과물을 약사님 개인 채널 콘텐츠로 활용
소통 채널 구조
  • 그룹 채널 — 전체 공지, 과제 결과물 공유, 자유 Q&A (게스트 + 호스트 전원)
  • 1:1 코칭 채널 (DM) — 게스트마다 전담 개발팀 코치 1명이 배정되어 개인 DM으로 직접 소통. 코드 막힘, 개인 도전 과제 방향성, 배포 이슈 등 개인별 상황은 이 채널에서 처리합니다.
  • 오프라인 모임 — 주 1회, 게스트 전원 + 호스트가 한자리에 모여 공통 주제 진행
💡 1:1 코칭이 필요한 상황 예시
• 그룹 채널에서 묻기 애매한 개인적인 막힘
• 본인의 도전 과제 아이디어 구체화·설계 상담
• 배포 환경/계정/인증 등 개인 환경 이슈
• 코드 리뷰 요청 (PR 또는 깃허브 링크 공유)
1
환경 세팅 & 실습 1·2 (캘린더 + 대시보드)
Week 1 · 오프라인 모임 (1h)
개발 환경을 세팅하고, 실습 1(캘린더)과 실습 2(대시보드)를 함께 시작합니다. Claude Code 기본 사용법과 스크린캡처를 활용한 바이브코딩을 체험합니다.
🤝 오프라인 모임 (1시간)
  • 실습 가이드 환경 세팅 함께 진행 (Node.js, Claude Code, Wrangler 설치)
  • GitHub에서 프로젝트 클론 + 개인 레포 연결
  • 로컬 서버 실행 — npx wrangler pages dev .로 브라우저에서 확인
  • Claude Code 기본 사용법: 스크린캡처 붙여넣기 + 자연어로 요청
  • 실습 1 (공구캘린더) & 실습 2 (공구대시보드) 함께 시작
📝 과제 — 다음 수업 전까지
  • 실습 1 (공구캘린더) 완료: 테마 색상 변경, 제품 매핑 추가, 자유 수정
  • 실습 2 (공구대시보드) 완료: 대시보드 커스터마이징 (차트, 컬럼, 통계 카드 등)
  • 각 결과물 스크린캡처/PNG로 그룹 채널에 공유
  • 실습 중 궁금한 점은 그룹 채널에 질문

2
실습 1·2 리뷰 & 실습 3 (당첨자폼) + 도전 과제 선정
Week 2 · 오프라인 모임 (1h)
1주차에 완료한 실습 1·2를 함께 리뷰하고, 실습 3(당첨자폼)을 빠르게 마무리합니다. 이번 주부터는 각자의 도전 과제를 선정하여 3·4주차에 발표 준비를 시작합니다.
🤝 오프라인 모임 (1시간)
  • 실습 1·2 과제 리뷰: 각자 커스터마이징한 캘린더/대시보드 발표 & 피드백
  • 핵심 개념 정리: REST API 인증(JWT), CORS 프록시, HTML 폼 + Apps Script
  • 실습 3 (당첨자폼) 함께 시작: 제품 정보 변경, 폼 필드 추가
  • 도전 과제 선정: 실습 가이드의 도전 과제(A~F) 중 택 1, 또는 본인이 만들고 싶은 주제
  • 실습 중 어려웠던 점 Q&A
📝 과제 — 다음 수업 전까지
  • 실습 3 (당첨자폼) 완료하기
  • 도전 과제 착수: 기획·요구사항을 정리하고 첫 프로토타입 시작
  • 진행 상황 또는 막힌 지점을 그룹 채널에 공유
실습 가이드: 당첨자폼 실습 바로가기 →
3
도전 과제 중간 리뷰
Week 3 · 오프라인 모임 (1h)
2주차까지 실습은 모두 마쳤습니다. 이번 주부터는 각자 선정한 도전 과제를 중심으로 진행 상황을 공유하고, 막힌 부분을 함께 풀어가는 코드 리뷰 시간을 갖습니다.
🤝 오프라인 모임 (1시간)
  • 도전 과제 중간 발표: 각자 진행 상황 공유 (인당 5~7분)
  • 막힌 부분 함께 디버깅: Claude Code 활용 팁 공유
  • 설계/UX 피드백: 호스트 + 동료 게스트의 코멘트
  • 다음 주 최종 발표 형식 안내 (시연 + 회고)
📝 과제 — 다음 수업 전까지
  • 도전 과제 완성: 받은 피드백을 반영하여 마무리
  • Cloudflare Pages에 배포하여 공유 가능한 URL 만들기
  • 최종 발표 자료 준비 (1~2장 분량 요약 또는 스크린캡처)

도전 과제 최종 발표 & 네트워킹
Week 4 · 오프라인 모임 (1h / 파이널)
4주간의 실습과 도전 과제 결과물을 발표합니다. 나만의 AI 코딩 활용 사례를 공유하고, 향후 활용 계획을 이야기합니다.
🤝 오프라인 모임 (1시간)
  • 도전 과제 최종 발표: 완성된 결과물 시연 + 배포 URL 공유 (인당 5~10분)
  • 4주간 가장 인상 깊었던 AI 코딩 경험 공유
  • 호스트 종합 피드백 및 수료
🍻 네트워킹
  • AI 활용 커뮤니티 채널 안내
