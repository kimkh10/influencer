1
환경 세팅 & 실습 가이드 실행
Week 1 · 오프라인 모임 (1h)
개발 환경을 세팅하고, 실습 가이드를 따라 프로젝트를 로컬에서 실행합니다. Claude Code 기본 사용법과 스크린캡처를 활용한 바이브코딩을 체험합니다.
🤝 오프라인 모임 (1시간)
  • 실습 가이드 환경 세팅 함께 진행 (Node.js, Claude Code, Wrangler 설치)
  • GitHub에서 프로젝트 클론 + 개인 레포 연결
  • 로컬 서버 실행 — npx wrangler pages dev .로 브라우저에서 확인
  • Claude Code 기본 사용법: 스크린캡처 붙여넣기 + 자연어로 요청
  • Q&A 및 문의 답변
📝 과제 — 다음 수업 전까지
  • 실습 1 (공구캘린더) 완료하기: 테마 색상 변경, 제품 매핑 추가, 자유 수정
  • 수정한 캘린더를 PNG로 저장해서 그룹 채널에 공유
  • 해보고 싶은 과제 제안: 앞으로 만들어보고 싶은 기능이나 페이지를 그룹 채널에 공유
  • 실습 중 궁금한 점은 그룹 채널에 질문
실습 가이드: 사전 준비 바로가기 →
2
캘린더 과제 리뷰 & 대시보드 실습
Week 2 · 오프라인 모임 (1h)
1주차 캘린더 과제를 리뷰하고, REST API 인증과 CORS 프록시 개념을 배웁니다. 실습 2(공구대시보드)를 함께 시작하고, 각자의 과제를 설정합니다.
🤝 오프라인 모임 (1시간)
  • 1주차 과제 리뷰: 각자 수정한 캘린더 공유 & 피드백
  • 실습 중 어려웠던 점 Q&A
  • 핵심 개념: REST API 인증(JWT), CORS 프록시 원리
  • 실습 2 (공구대시보드) 함께 시작: 대시보드 접속 & 로그인
  • 이번 주 과제 설정: 대시보드 기반 자기만의 과제 선정
📝 과제 — 다음 수업 전까지
  • 실습 2 (공구대시보드) 완료하기: Swagger 확인, 대시보드 커스터마이징
  • 자기만의 과제 1개 선정하여 완성 (예: 차트 변경, 컬럼 추가, 통계 카드 등)
  • 결과물 스크린캡처와 함께 그룹 채널에 공유
실습 가이드: 공구대시보드 실습 바로가기 →
3
대시보드 과제 리뷰 & 당첨자폼 실습
Week 3 · 오프라인 모임 (1h)
2주차 대시보드 과제를 리뷰하고, HTML 폼 구조와 Google Apps Script 연동을 배웁니다. 실습 3(당첨자폼)을 함께 시작하고, 도전 과제를 설정합니다.
🤝 오프라인 모임 (1시간)
  • 2주차 과제 리뷰: 각자 커스터마이징한 대시보드 발표 & 피드백
  • 실습 중 어려웠던 점 Q&A
  • 핵심 개념: HTML 폼 요소, Google Apps Script 연동
  • 실습 3 (당첨자폼) 함께 시작: 제품 정보 변경, 폼 필드 추가
  • 도전 과제 선정: 실습 가이드의 도전 과제(A~F) 중 택 1
📝 과제 — 다음 수업 전까지
  • 실습 3 (당첨자폼) 완료하기: 제품 변경, 필드 추가, 관리자 페이지 확인
  • 도전 과제 1개 완성하기 (최종 발표 준비)
  • 결과물을 배포 URL과 함께 그룹 채널에 공유
실습 가이드: 당첨자폼 실습 바로가기 →
4
도전 과제 발표 & 네트워킹
Week 4 · 오프라인 모임 (1h / 파이널)
3주간의 실습과 도전 과제 결과물을 발표합니다. 나만의 AI 코딩 활용 사례를 공유하고, 향후 활용 계획을 이야기합니다.
🤝 오프라인 모임 (1시간)
  • 3주차 당첨자폼 과제 리뷰
  • 도전 과제 발표: 각자 완성한 도전 과제 시연 (인당 5~10분)
  • 4주간 가장 인상 깊었던 AI 코딩 경험 공유
  • 호스트 종합 피드백 및 수료

  • AI 활용 커뮤니티 채널 안내
