AI 그룹 스터디 커리큘럼 페이지(`study-curriculum.html`)를 생성하거나 수정합니다.

## 개요
4주 AI 스터디 프로그램의 커리큘럼 안내 페이지입니다. 오프라인 미팅 + 과제 수행 방식으로 운영됩니다.

## 파일 구조
| 파일 | 역할 |
|------|------|
| `study-curriculum.html` | 커리큘럼 안내 페이지 |
| `study.html` | 스터디 신청 폼 |
| `study-admin.html` | 스터디 관리자 대시보드 |

## 현재 커리큘럼 (4주 과정)

### 운영 구조
- 호스트 3명 + 게스트 6명 (최대 9명)
- 오프라인 1시간 미팅 (Q&A + 과제 플래닝 + 문의 답변)
- 다음 수업 전까지 과제 수행

### 주차별 내용
- **Week 1**: 환경 세팅 + 프로젝트 다운로드/실행/배포 (git clone, npx serve, Cloudflare Pages)
- **Week 2**: AI 이미지 생성 & 콘텐츠 제작 (DALL-E, Midjourney, 카드뉴스, 포스터)
- **Week 3**: 공동구매 캘린더 & 데이터 시각화 (Google Sheets API, html2canvas, 관리자 대시보드)
- **Week 4**: 최종 발표 & 네트워킹

## 작업 시 참고사항

1. **단일 파일**: `study-curriculum.html`에 CSS + JS 모두 포함
2. **오렌지 그라데이션 테마**: study.html과 동일한 스타일
3. **반응형**: 모바일 480px 브레이크포인트
4. **하단 CTA**: 신청 폼(`./study.html` 상대 경로) 연결

## 사용자 요청에 따라

- **커리큘럼 내용 수정**: 주차별 미팅 안건, 과제 내용 변경
- **주차 추가/제거**: 커리큘럼 구조 변경
- **운영 구조 변경**: 호스트/게스트 인원, 미팅 방식 수정
- **디자인 변경**: 카드 레이아웃, 색상, 애니메이션 수정
- **신청 폼 수정**: study.html 폼 필드 추가/변경
- **관리자 대시보드 수정**: study-admin.html 기능 변경

수정 완료 후 사용자에게 로컬 테스트(`npx serve .`) 또는 배포(`/deploy`) 안내를 제공하세요.
