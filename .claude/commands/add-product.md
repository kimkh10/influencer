# 제품 매핑 추가

새로운 제품의 이름/설명/배지 매핑을 index.html에 추가합니다.

## 필요 정보
사용자에게 아래 정보를 확인합니다:
1. **스프레드시트 타이틀 키워드** (예: "프로바이오틱스")
2. **표시 이름** (예: "분홍이")
3. **설명** (예: "인기템 깜짝공구\n단 3일!")
4. **배지** (예: "HOT", "품절대란템", 또는 없음)

## 수정 위치
- `getShortName()` — 키워드 → 표시 이름 매핑 추가
- `getProductDesc()` — 키워드 → 설명 매핑 추가
- `getBadge()` — 키워드 → 배지 매핑 추가 (있는 경우만)

## 수정 후
Vercel 배포: `vercel --yes --prod`
